Bridge Builder Help

Introduction

Bridge Builder is a program that allows you to build structures and then test them to destruction for fun and enlightenment. There are a number of pre-set challenges and a free-form construction mode in which you can build whatever you wish to within the limits of our materials and your imagination.

These controls allow you to switch modes:

Key Icon Function

Esc   Cancel (in the midst of some operations)

(Note: Load/Save invoke a dialog but still allow a shortcut ‘n’ to file slots 0-9)

Edit Mode

Edit mode allows you to create your structure by selecting materials, positioning Segments that are connected at Bolts and using a variety of tools and macros. Save your structure to disk or load previously saved structures.

There are a number elements displayed in Edit mode

Display

The display shows the landscape of the Bridge Builder world. In Edit mode there is a grid and toolbox - both of which can be turned on or off. The Toolbox can be re-positioned by dragging it by the title bar.

Zooming the Display

The display can zoom in and out to show either the big picture or more detail for fine work. The Grid will show at various levels right down to the smallest construction unit.

Edit Controls

  • Single Click Select nearest Segment or Bolt (and attached Segments)
  • Drag Box (left) Selection box of Bolts and Segments (if either Bolt is selected)
  • Drag Box (right) Selection box of Bolts and Segments (if both Bolts are selected)
  • Control Selection Extend selection
  • Alt Selection Group selection
  • Del Delete selection of Bolts and Segments (Note: Bolts are only deleted if there are no connected Segments)
  • C Copy selection to cursor (Note: Bolts with the same position are merged)
  • V Move selection to cursor
  • B Stretch selection to cursor
  • O Place copy/move
  • X Flip selection horizontally
  • Y Flip selection vertically
  • M Cycle through materials: Wood, Steel, Concrete, Cable etc
  • . Toggle Grid display
  • Tab Toggle tool panel On/Off (in Edit mode only)

Tool Icons

  • Select Click to select a single Bolt or Segment. Click and drag to select using a rectangular marquee. Drag box Left to select segments with either Bolt in rectangle (Green marquee). Drag box Right to select only Segments with both Bolts in the rectanlge (Red marquee). Hold down [Ctrl] and make further selections to extend selection. [Alt]-Select to select a group of Segments placed by a single Copy/Move/Stretch or macro operation.
  • Copy
  • Move
  • Stretch
  • Build Build a single segment. Click on the starting point and drag to build a segment in the current primary material. Attaches to nearby bolt if one exists. As you move the endpoint, the line will be Green if you can validly place the endpoint there or Red if you cannot. Reasons for why you cannot place an endpoint - It would make the Segment longer than is allowed for that material. - The endpoint would be placed in an exclusion region
  • Build Multiple Build a segment and start a new one until cancelled (Esc or other selection)
  • Group
  • Replace
  • Line Build a line of one or more segments to connect two points
  • Beam Build a straight truss structure between the selection points
  • Arch Build a parabolic truss structure within the selection box
  • Circle Build a circular truss structure with centre and radius according to the selection points

Materials

Material Image Properties Description
Wood Beam
Weight 10
fTens 5
fComp 6
Cost 2
Wood is light and strong for its weight but not able to carry large loads. Used for relatively small structures or for low load bearing areas in more ambitious structures
Steel Beam
Weight 100
fTens 800
fComp 1250
Cost 50
Steel is a strong and versatile material the introduction of which revolutionised construction from the 18th Century. It remains a primary construction material in general use and is often the main load bearing material where primarily tension or mixed tension/compression forces must be withstood.
Concrete Beam
Weight 600
fTens 450
fComp 4380
Cost 200
By combinding the strength in compression of cement (which was known to the ancients) and embedded re-inforcing rods of mild steel composition to resist tension forces, Concrete achieves an extremely useful balance of properties and has become one of the most commonly used materials in modern construction. It is not well suited to primarily tensile loads and also has a high unit weight making selective placement the key to successful use.
Steel Cable
Weight 20
fTens 600
fComp 0
Cost 150
Mild steel wires spun into multi-stranded cables revolutionised bridge building in the early 20th Century. With almost no strength in compression but with unmatched tensile strength and previously unheard of strength to weight ratios, Steel Cables liberated designers from the massive structural designs that had previously been necessary. Steel Cables find a variety of uses beyond the most obvious application in suspension bridges.
Boron Monofilament
Weight 10
fTens 5000
fComp 0
Cost 600
Boron Monafilament Cables or more correctly Boron-Boron composite tensile materials became available in the early 21st Century and heralded the beginning of a new golden age of structural engineering. While slightly less extraordinary than Carbon-Carbon composites, Boron Monofilaments have proven to be entirely more practical to manufacture and deploy in real-world applications. Their very high cost however means that they are still only used when truly exceptional properties are called for and no other material will do.
